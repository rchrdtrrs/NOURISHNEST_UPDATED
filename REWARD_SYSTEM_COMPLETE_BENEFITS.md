## 💵 Point System

### How Users Earn Points
```
Cook a Recipe          →  +10 points
Complete Meal Log      →  +5 points
Rate a Recipe          →  +3 points
Invite a Friend        →  +50 points (bonus)
Complete Daily Challenge → +25 points
Share Achievement      →  +15 points
```

### Point Redemption Chart
```
Advanced Analytics     ← 100 pts
AI Substitutions       ← 100 pts
Chef Recipe (each)     ← 75 pts
Theme (each)           ← 50 pts
Badge (each)           ← 30 pts
```
